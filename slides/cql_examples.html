CQL, Cassandra Query Language

 

  • Offre une perception semblable au SQL
  • Une représentation des données familières en colonnes-lignes
  • Beaucoup plus intuitif que l'interface basé sur Thrift

Un petit exemple...

 


CREATE TABLE t (k int PRIMARY KEY, other text)
CREATE TRIGGER myTrigger ON myTable USING 'org.apache.cassandra.triggers.InvertedIndex';
UPDATE UserActions SET total = total + 2 WHERE user = B70DE1D0-9908-4AE3-BE34-5573E5B09F14 AND action = 'click';
