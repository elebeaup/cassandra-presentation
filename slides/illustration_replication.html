- Données répliquées aux noeuds 1, 2 et 3.

Stratégies de réplications

Détermine où les noeuds de réplications sont placés

- SimpleStrategy : Pour les data centers esseulés

- NetworkTopologyStrategy

Niveaux de consistance

LOCAL_QUORUM : requiert un quorum des noeuds dans le même DC que le coordinateur :

- besoin d'un DC local disponible avec un quorum des réplications

- Consistance locale

- Les écritures sont toujours répliquées dans tous les DC

EACH_QUORUM : requiert un quorum des noeuds dans chaque DC :

- Fonctionne uniquement si tous les DC sont disponibles avec un quorum des réplications

- Consistance globale sur tous les DC

- ex : un DC backup qui doit être consistant avec un autre DC
